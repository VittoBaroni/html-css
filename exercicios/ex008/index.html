Tipos de Links:

Links Externos:

Links externos nada mais são do que links que levam para outro site, como por exemplo:


Em Game of Thrones, meu personagem favorito é a Arya


Como você pode ver, clicando em cima do nome "Arya", você é redirecionado para outra pagina, explicando sobre ela. Porém, é interessante fazer com que o link que voce clicou seja redirecionado para uma nova guia dentro do seu navegador. Exemplo:


Em Game of Thrones,meu personagem favorito é a Arya


Agora, como pode ver acima, o link está levando à uma nova guia do navegador 😃
